# osep-scripts
Scripts to ease my osep workflows.

This project is WIP and i'll keep posting new scripts here as I work through my course.

## Disclaimer
All testing has been done on ubuntu 24.04. Scripts seem to behave differently on kali.

## Scripts
### update-hosts.sh
wrote this quick script to take in a CIDR range and add the host name and the IP address to your /etc/hosts file. (awk fields need to be changed if using kali)

### xor.py:
wrote this script so I can quickly xor encode my shellcode generated by msfvenom/sliver using an alphanumeric key of my choosing. Today it accepts the following arrays: c, cs, ps1, and vba. It also encodes raw binary shellcode.

Example usage: `python3 xor.py -t cs -i rev64.cs -k z --raw | clip`

The above command will xor a csharp style shellcode with the key "z" and output nothing except the formatted xor'd array which I passed to clip (alias around xlip)
